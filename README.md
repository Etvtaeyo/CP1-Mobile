#  Formulário de Cadastro Acadêmico

Este é um aplicativo desenvolvido em **React Native** que permite ao usuário preencher um formulário com informações acadêmicas e visualizar os dados cadastrados na tela.

## Funcionalidades

* Inserção de dados:

  * Nome completo
  * Curso
  * Disciplina
  * Descrição pessoal

## Tecnologias Utilizadas

* React Native
* Expo
* JavaScript (ES6+)
* Hooks (`useState`, `useEffect`)
* Biblioteca de ícones

## Interface

O aplicativo possui:

* Cabeçalho com título e descrição
* Formulário estilizado
* Botões de ação (Enviar / Limpar)
* Área de resultado com feedback visual


## Lógica do Aplicativo

* Os dados são armazenados usando o hook `useState`
* O botão **Enviar** valida se todos os campos estão preenchidos
* Se válido, os dados são exibidos na tela
* O botão **Limpar** reseta todos os campos e remove os dados exibidos
